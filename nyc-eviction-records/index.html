How it works

Check a building in 10 seconds.

01

Enter the address

Any NYC address. We search for it directly in the city's eviction records database via NYC Open Data.

02

Read the history

Date, unit, eviction type, and executing marshal for every filing. See the pattern at a glance.

03

Use it before you sign

A persistent pattern of evictions is a signal. Bring it to a lease conversation, or cite it in a hearing.

Where the data comes from

All public. All cited.

Primary

NYC DOI Evictions · Open Dataset

Department of Investigation. Executed residential evictions by city marshals, updated weekly (6z8x-wfk4).

API

NYC Open Data · Socrata

All queries made live from your browser to the official NYC Open Data endpoint. No middleman, no caching.

Coverage

Executed evictions only

This dataset covers marshal carry-outs — the final stage of the eviction process. It does not include all court filings or cases settled before execution.

Reference

HPD Registration Contacts

Cross-linked so any eviction record can be traced back to the building's current ownership.
