How it works

Three clicks. No PDFs.

01

Type an address

Street and borough. We query HPD's database directly — live, not cached.

02

Read the record

Open violations, severity bands, and inspection dates — summarized on one screen, in plain language.

03

Share or cite

Every data point comes from the official HPD record via NYC Open Data. Copy a link and hand it off.

Where the data comes from

All public. All cited.

Primary · Violations

HPD Housing Maintenance Code Violations

NYC Dept. of Housing Preservation & Development. Open dataset, updated daily via NYC Open Data (wvxf-dwi5).

API

NYC Open Data · Socrata

All queries are made live from your browser to the official NYC Open Data endpoint. No middleman, no caching.

Secondary

HPD Registration Contacts

Building owner and managing agent filings — used to resolve LLCs where possible.

Reference

PLUTO · Property Land Use Tax Lot Output

NYC Dept. of City Planning — used to cross-reference ownership and address resolution.
